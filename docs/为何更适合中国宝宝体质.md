# 为何更适合中国宝宝体质

lingti-bot 更适合中国用户的几个原因：

## 1. 国内平台优先支持

| 平台 | 状态 |
|------|------|
| 飞书/Lark | ✅ 已支持 |
| 钉钉 | 🚧 开发中 |
| 企业微信 | 🚧 开发中 |
| 个人微信 | ✅ 已支持（云中继） |

而非只支持 Slack/Discord 等海外平台。

## 2. 中文语音输入默认

- whisper 默认 `-l zh` 中文识别
- 不需要额外配置语言
- 开箱即用的中文语音交互

## 3. 云中继模式

- 解决国内网络环境问题
- WebSocket 穿透，无需公网 IP
- 微信等平台无需自建服务器

## 4. 本地化日历集成

- 修复了 macOS 日历的中文日期格式问题
- 使用日期构造器而非依赖系统 locale
- 完美支持中文系统环境

## 5. 中文文档

- README 全中文
- 命令示例贴合国内使用场景
- 无需翻译即可上手

## 6. 一键安装

```bash
curl -fsSL https://cli.lingti.com/install.sh | bash -s -- --bot
```

## 对比海外同类产品

| 特性 | lingti-bot | OpenClaw 等 |
|------|-----------|-------------|
| 飞书支持 | ✅ | ❌ |
| 钉钉支持 | 🚧 | ❌ |
| 企业微信 | 🚧 | ❌ |
| 微信接入 | ✅ | ❌ |
| 中文语音 | ✅ 默认 | 需配置 |
| 中文文档 | ✅ | ❌ |
| Slack | ✅ | ✅ |
| Discord | ✅ | ✅ |

**简单说**：海外类似工具主要服务 Slack 生态，而 lingti-bot 从一开始就为飞书、钉钉、企业微信设计。
